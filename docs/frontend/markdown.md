# Markdown 常用语法使用方法

## 1. 标题

用 `#` 表示标题，`#` 数量越多级别越低（最多 6 级）。

```md
# 一级标题
## 二级标题
### 三级标题
```

## 2. 段落与换行

- 段落之间空一行。
- 行末加两个空格可换行，或使用 `<br>`。

```md
这是第一段。

这是第二段。  
这是同一段内换行后的内容。
```

## 3. 强调（粗体/斜体/删除线）

```md
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体***
~~删除线~~
```

## 4. 列表

### 无序列表

```md
- 项目 A
- 项目 B
  - 子项目 B1
```

### 有序列表

```md
1. 第一项
2. 第二项
3. 第三项
```

## 5. 引用

使用 `>` 表示引用，可嵌套。

```md
> 这是引用内容
>> 这是二级引用
```

## 6. 代码

### 行内代码

```md
使用 `console.log()` 输出信息。
```

### 代码块

使用三个反引号包裹，可指定语言高亮。

````md
```js
function greet(name) {
  return `Hello, ${name}`;
}
```
````

## 7. 链接

```md
[OpenAI](https://openai.com)
```

## 8. 图片

```md
![图片说明](https://example.com/image.png)
```

## 9. 分割线

可使用 `---`、`***` 或 `___`。

```md
---
```

## 10. 表格

```md
| 姓名 | 年龄 | 城市 |
| --- | --- | --- |
| 张三 | 20 | 北京 |
| 李四 | 22 | 上海 |
```

## 11. 任务列表

```md
- [x] 已完成任务
- [ ] 未完成任务
```

## 12. 转义字符

在特殊字符前加反斜杠 `\`。

```md
\*这不是斜体\*
```

## 13. HTML 混写（可选）

Markdown 中通常可直接使用部分 HTML 标签。

```md
<kbd>Ctrl</kbd> + <kbd>C</kbd>
<br>
```

---

以上内容覆盖了 Markdown 的常用语法，适合日常文档、笔记和 README 编写。
